# 11ty ejs template 2022

## Usage （使い方）

Use npm scripts as below,
you can build html files from ejs files.

```shell
npm run build
```

View on Browser

```shell
npm run dev
```

Lint EJS

```shell
npm run lint
```

## What you can learn with this template

- 11ty with ejs
- ejs-lint
- Prettier
